// OpenLayers Interactive Map Project

An interactive web map built using **OpenLayers**, **HTML**, **CSS**, and **JavaScript** — featuring zoom controls, home view, current location, drawing tools, and GeoJSON export.

---


**Home Button** – Reset map view to India (default position).  
**Zoom In / ➖ Zoom Out** – Manually control zoom levels.  
**Current Location** – Locate and zoom to your GPS position.  
**Draw Point** – Add markers or single locations.  
**Draw Polygon** – Draw areas directly on the map.  
 **Export to GeoJSON** – Save drawn features as a `.json` file.   **Simple Toolbar UI** – Clean and responsive layout using CSS.
